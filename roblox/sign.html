Instructions for patching

By MAO
  1. Get HxD
  2. Open RobloxPlayerBeta.exe
  3. Ctrl + F "bgiaaac"
  4. Select the entire thing
  5. Replace it with the public key
  6. Save it as a new executable, and you're done
